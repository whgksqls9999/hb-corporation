# Reflect — 서기 호출 (수동 트리거)

기록의 실체는 **scribe-manager 부서**가 수행한다. 이 스킬은 그 수동 호출 래퍼다.
(팀 작업 후에는 Stop 게이트가 자동으로 서기를 강제하므로 보통 직접 부를 필요 없다.)

## 절차
1. Agent 도구로 `scribe-manager`를 스폰한다.
2. task에 다음을 실어 보낸다:
   - 이번 작업의 맥락 요약 (무엇을 했나)
   - qa-manager 판정 (있으면 원문 인용)
   - 채택/revert 여부 등 객관 사실
3. 서기의 보고(저장된 카드 이름)를 사용자에게 전달한다.

판정·저장·장부 청산의 기준은 전부 scribe-manager 정의를 따른다 (여기서 재정의하지 않는다).
